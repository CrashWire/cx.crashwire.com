Chron X Utilities
DeckAlyz

Current version: 3.61 (what's new?)

Last Program Update: January 24th, 2002.
Last Card List Update: May 30th, 2002. (Breakpoint: Monastery)

Description: DeckAlyz is a utility for analyzing Chron X decks. It provides many statistics on the decks, and can find several problems with it's design, such as insufficient resources or high chance of hands without assets/bases. It includes a card database that can be updated with either CXlist.exe (downloadable from the main Chron X website) or the Cards.zip file downloadable here.

The current version supports all cards from First Edition up to Breakpoint and may support further expansions if no rule changes are introduced.

How to get DeckAlyz running:

  1. Download the zip file below.
  2. Unzip it so that all files in the zip end up in the same directory.
  3. Run DeckAlyz.
  4. If there are any problems whatsoever, e-mail CrashWire at crashwire@crashwire-x.com or Birdwatcher at eytanzw@yahoo.com.
Download here:

deckalyz.zip (345K zipped, 816K when opened)
cards.zip (58K zipped, 211K when opened)


CXMini

Current Version: 2.2 (what's new?)

Last Update: July 27th, 2000.

Description: Originally created by Neil T. Ormiston, this utility allows you to keep track of your drawn cards during a game and helps prevent many common mistakes.

How to get CXmini running:

  1. Download and unzip file below
  2. Run CXMini.
  3. If there are any problems whatsoever, e-mail Birdwatcher at eytanzw@yahoo.com.
  4. For additional (optional) features, keep it in the same directory as DeckAlyz.

Deck Generator
Note: This program is currently frozen.

Current Version: 1.01

Last Update: February 1st, 2000.

Description: This small utility allows you to generate random Chron X decks out of a card selection. You set how many assets, bases, and other cards to place in, and can set additional limits (no Very Rares, no assets with FP under 3, etc.).

The deck generator requires DeckAlyz to run, and must be placed in same directory as DeckAlyz.

How to get Deck Generator running:

  1. Download zip file below
  2. Unzip it into the same directory as DeckAlyz.
